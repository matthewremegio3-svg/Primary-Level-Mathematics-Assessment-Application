Primary-Level Mathematics Assessment Application 

A simple and fun Python quiz game with cute cat reactions, sound effects, and multiple difficulty levels.

Features:
• Cat reacts (neutral, excited, sad)
• 10 lives with heart icons
• Sound effects
• Hint system
• Easy, Medium, Hard levels
• Saves results to quiz_results.csv
• Responsive UI (scales when resizing window)

Tech Used: 
• Python (Tkinter)
• Pillow (image handling)
• Pygame (sound effects)
• JSON & CSV (data storage)

How to Run:

Install requirements:

pip install pillow
pip install pygame

Run the app:

python catquizapp.py

Important Files

catquizapp.py — main program

quiz_level.json — quiz questions

quiz_results.csv — auto-saved results

Cat images & heart icons

Sound files (correct.wav, wrong.wav, click.wav)
